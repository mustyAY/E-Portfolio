  • Countries of Earth

    Html, CSS, JavaScript

    I used a REST Countries API to create this project. I basically wanted to create something a bit educational, fun, and very easy to navigate. It displays the flags of all the countries and shows the name of the country when you hover on a flag. When you click on a country or search it in the search bar it shows some essential details about the country. Now this API is not the best out there but I utilize CSS heavily to show error messages where I stumbled on problems I couldn't fix with vanilla JavaScript and there is also a better version of this project I have created with React on my E-portfolio.

  • Firebase CRUD Operations

    Html, CSS, JavaScript, React, Firebase

    Now I know this is not the prettiest application but it's the functionality that matters, okay it's not so functional because everything is hard coded so you have to read the code to understand what's going on lol..
    So I basically Registered a user with email "email@email.com" and this user can create, read, update, and delete (i.e CRUD) posts which get stored in the firebase firecloud.
    The Login button logs the user in and the round button with E which is the first letter of the user's email logs the user out.
